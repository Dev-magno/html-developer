Extrutura HTML

Voltar

Tudo que estiver entre as tags <html></html> é um documento html. o html consiste em duas partes: 1) cabeçalho ou head representado pelas tags <head></head>. Tudo que é colocado entre essas tags será carregado antes de ser exibido para o usuário. 2) Corpo ou body que é representado pela tag <body></body>.Tudo que estiver aqui irão aparecer visualmente para o usuário. No head tem uma tag chamada TITLE que é o título da página a ser usada <title></title>. Esse título aparece na aba do navegador.

O copor, <body></body>, é o local onde todo o site será desenvolvido. Todas as tags para a sua confecção devem ser colocadas entre essas tags que por sua vez se encontram no documento html. É preciso atenção para não tentar, por exemplo, colocar a tag head dentro do body, o que constituíria uma falha na estrutura do HTML.

Tags e seu funcionamento

Voltar

A tag <I></I>. Tudo que estiver entre essa tag o navegador interpreta como itálico. O a palava HTML ela está entre a tag <I></I> que a deixou em itálico. A tag <strong></strong> faz com que o texto fique em negrito. Novamente veja a palavra HTML, note que ela está em negrito porque foi colocada entre as tags <I></I>.

A gora se você desejar colocar a palavra HTML em negrito e itálico, desta forma: HTML basta colocar a tag dentro da tag e o texto entre as tag <I></I> e o resultado esperado é que ela fique em negrito e itálico como no exemplo acima. É importante frisar que há muitas outras tags. Você poderá fazer uso de sites específicos para aprender sobre elas.

Atributos das Tags

Voltar

Atributos são propriedades que uma tag pode ter. Existem atributos que são mais globais e tem outros que são expecíficas de cada tag. Exemplos: Na tag <strong id = "titulo"</strong> pode te um atributo chamado id serve para identificar no JavaScript qual elemento você quer mexer. Significa que o id é um identificador. Há também o style que é usado pelo CSS para formatar diretamento o elemento. Essa fortamação seria por exemplo: colocar outra fonte, colocar uma determinada cor, alinhar o texto etc. Outro atributo é o Class usado também CSS para padronização do conteúdo do site.

o input, <input>, tem um atributo chamado type <input type = ""> que é do tipo do tipo específico dele mesmo. Nesse atributo pode determinar que o tipo do input sera um campo de texto. Fica assim: <input type = "text">. Nesse inpute somente texto será aceito. Ou pode ser do tipo número, dessa forma: <input type = "number">. Nesse input só será aceito o tipo número. Textos

Voltar

As tags <h1> a <h6> -> são títulos no HTML. Eles vão de distinguindo em Título <h1></h1> e subtítulos <h2></h2> em sequência. Se você vai fazer um arquivo de um livro, por exmeplo, onde há título e sutítulos essa tag é adequada para ser usada. Uma das tags bastante usada é tag <p> a </p> que serve para escrever parágrafos no documento HTML. Ela será usada em todas as páginas que forem escritas usando essa linguagem de marcação de texto.

Temos também a tag <blockquote></blockquote>. Ela é usada para fazer uma citação. Por exemplo, se você estiver fazendo uma página para disponibilizar um conteúdo e quizer citar um estudioso, então usa-se essa tag. Vejamos um exemplo:

"A Internet é baseada em HTML, o que é realmente uma coisa muito simples. É uma linguagem de marcação muito simples. Não é uma linguagem de programação, é uma linguagem de marcação. Isso é importante. As pessoas que não a entendem podem pensar que está fazendo mais do que realmente está." - Douglas Crockford
Veja que o texto acima ficou visualmente diferente dos demais, porque está dentro da tag <blockquote></blockquote>.

Temos mais uma tag a ser vista dentre mmuitas outras, é a tag que serve para colocar um texto (como asterico) na linha de cima do texto de outra tag ou uma nota de rodapé. Nesse sentido, ela é muito útil.

Listas

Voltar

Em HTML, as listas são usadas para organizar e apresentar informações de forma estruturada. Existem dois tipos principais de listas: listas ordenadas e listas não ordenadas. As listas ordenadas são feitas com a tag <ol></ol>.
São usadas para criar uma lista de itens ordenados numericamente ou alfabeticamente.
Os itens são automaticamente numerados ou marcados com um formato específico, como números, letras ou romanos.
São ideais para sequências lógicas de itens ou instruções passo a passo.
Aqui está um exeplo de lista ordenada:

  1. Maçã
  2. Maracujá
  3. Uva
As listas não ordenadas são feitas com a tag <ul></ul>. São usadas para criar uma lista de itens sem uma ordem específica. Os itens são marcados com bullets padrão, mas o estilo pode ser alterado com CSS. São úteis para listar itens que não têm uma ordem sequencial ou para destacar uma série de pontos importantes.
Veja um exemplo de lista não ordenada:
  • Rio de Janeiro
  • São Paulo
  • Brasília

Ambos os tipos de listas <ol> e <ul>são construídos usando tags de lista <li></li>, que representam cada item da lista. Essas listas podem ser aninhadas, ou seja, uma lista pode conter outra lista omo um de seus itens.

  • Frutas:
    • Maçã
    • Banana
    • Uva
  • Cidades do Brasil:
    • Rio de Janeiro
    • São Paulo
    • Brasília

Neste exemplo, a lista principal é uma lista não ordenada <ul> que contém dois itens. Cada item da lista principal contém uma lista aninhada de itens, uma lista de frutas e uma lista de cidades do Brasil, respectivamente, dentro de elementos <ul> aninhados. Cada uma dessas listas aninhadas é composta por elementos <li> que representam os itens da lista. Elas são uteis também para fazer menu e que geralmente é usado em páginas HTML.
